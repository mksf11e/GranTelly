# Public Release Notes

This repository is intentionally sanitized.

The public product name is **GIMY**, short for **Grandma I Miss You**. Internal version labels used during development are not part of the public positioning.

## Removed From The Public Version

- Real user memories and family summaries.
- Real OpenClaw agent prompts.
- Real conversation sessions and trajectory logs.
- Real photo assets from private demos.
- Telegram token, chat ID, pairing data, and allowlists.
- VPS IP addresses, gateway tokens, SSH details, and deployment state.
- Production plugin memory data and vector stores.

## Kept In The Public Version

- High-level architecture.
- ESP32 WebSocket protocol.
- Mock device and mock backend flow.
- Sample metadata schemas.
- Display image preparation example.

## Public Positioning

This is not a medical monitoring product and not a replacement for family care. It is a dual-agent communication aid designed to reduce friction in intergenerational connection and encourage more regular family conversations.
